Running Wemp for the first time

When you try to run Wemp, Windows SmartScreen might show a warning because the application is not signed with a certificate.

  1. When the warning appears, click "More info"
  2. Click "Run anyway" to start Wemp
  3. This warning will only appear the first time you run Wemp
